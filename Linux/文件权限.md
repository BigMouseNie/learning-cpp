# 权限

## 权限表说明

| 宏        | 含义         | 八进制 |
| --------- | ------------ | ------ |
| `S_IRUSR` | 用户读权限   | 0400   |
| `S_IWUSR` | 用户写权限   | 0200   |
| `S_IXUSR` | 用户执行权限 | 0100   |
| `S_IRGRP` | 组读权限     | 0040   |
| `S_IWGRP` | 组写权限     | 0020   |
| `S_IXGRP` | 组执行权限   | 0010   |
| `S_IROTH` | 其他读权限   | 0004   |
| `S_IWOTH` | 其他写权限   | 0002   |
| `S_IXOTH` | 其他执行权限 | 0001   |
